Features

Everything you need

No more terminal windows everywhere. One app to rule all your processes.

📊

Live Dashboard

See all your processes at a glance. Real-time CPU, RAM, uptime, and sparkline charts update every 2 seconds.

🔄

Auto-Restart on Crash

Configurable per-process with exponential backoff. Set max retries, delay, and multiplier so crashes don't kill your server at 3am.

📋

Live Log Viewer

Stream stdout/stderr in real-time with ANSI color rendering. Search, filter by log level, export to file, and send console commands.

✦

AI Crash Analysis

When a process crashes, click "AI Analyze" to get a clear explanation of what went wrong and exactly how to fix it. Powered by Groq, OpenRouter, or Anthropic.

⛏

Minecraft RCON

Built-in RCON client for live player counts, in-dashboard command palette, and graceful server shutdown with the Minecraft stop command.

🌐

playit.gg Tunnels

No port-forwarding needed. Enable playit.gg per Minecraft server and the tunnel URL appears right on the dashboard card.

⏰

Scheduler

Set per-process start and stop times with day-of-week selection. Schedule your server to run only during active hours.

📈

Stats & History

24-hour CPU, RAM, and player count charts drawn natively with canvas. View crash counts, uptime percentage, and availability stats.

🖥

System Tray

Minimizes to tray instead of closing. Tray icon shows green, yellow, or red based on process states. Start/stop processes without opening the window.

How it works

Up and running in minutes

No servers, no subscriptions, no accounts. Just download and go.

1

Install

Download the Windows installer and run it. That's it — no runtime, no dependencies.

2

Add a process

Click "Add Process", pick a folder, and let AI auto-detect how to start it — or fill in the command yourself.

3

Hit Start

Your bot or server is running. Live logs stream in, CPU and RAM update in real-time, and auto-restart is already on.

4

Never worry again

If it crashes, Warden restarts it automatically — and AI can tell you exactly why it crashed.
